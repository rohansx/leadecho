How it works

1

FAQ

Common questions

Does it work without API keys?

Reddit and Hacker News work completely without any API keys — they use public JSON endpoints. For AI scoring (intent classification, relevance scores) you need to provide your own key from OpenAI, Anthropic, or a compatible provider. Twitter and LinkedIn require browser session cookies for authenticated access.

How is this different from keyword alerts?

Keyword tools match exact strings. LeadEcho uses sentence embeddings to match by meaning against your pain-point profiles. Someone asking "how do I stop losing clients to bigger agencies" won't match a keyword alert for "agency management software" — but LeadEcho will score it as a hot lead if that's your ICP pain point.

How does the reply posting work — won't I get banned?

The Chrome extension types replies using a human-mimicry engine: Gaussian keystroke timing (~80ms ±40ms per character), punctuation pauses, and a 2–8 second pre-engagement delay before typing starts. It uses the platform's own input system (not API calls), so it behaves like a human typing in the browser. You review and approve every reply before posting — nothing is fully automated.

What AI providers are supported?

LeadEcho is BYOK (Bring Your Own Key). It supports OpenAI, Anthropic, and any OpenAI-compatible endpoint (Groq, Together AI, local Ollama, etc.). Voyage AI is used for embedding-based semantic matching, but a fallback keyword-only mode works without it.

How much does AI scoring actually cost?

Very little. Each mention uses one LLM call (~200 tokens). At 2,000 mentions/month with GPT-4o-mini, that's ~$0.05. Embeddings cost ~$0.002 per 1,000 mentions. Most users spend under $2/month in API costs for the AI features.

Can I use it without the Chrome extension?

Yes. The backend monitors platforms on its own schedule (every 5 minutes). The extension adds passive signal collection and the reply queue side panel, but the core inbox, scoring, and AI drafts all work from the web dashboard alone.

Is LinkedIn monitoring reliable?

LinkedIn is the hardest platform to monitor. LeadEcho uses Pinchtab (a browser automation sidecar) with your own li_at session cookie to search LinkedIn as you. For more stealth, the Pro tier includes Camoufox (a fingerprint-spoofing Firefox). You'll need to occasionally refresh session cookies as they expire — the dashboard has a browser sessions page for this.
